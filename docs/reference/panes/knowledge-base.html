Knowledge Base

Live Pane type

Project knowledge entities with freshness indicators. Click to open the markdown file. Quick-search via senkani_knowledge.

Authorship badge

Every entity row carries an AI / Human / Mixed / Untagged capsule (Phase V.5). The save path prompts before persisting an ambiguous row, so a tagged badge means the operator chose. Legacy rows that predate V.5 render the subdued "Untagged" affordance with a tooltip pointing at senkani authorship backfill; nothing is silently inferred.

Keyboard

Open via ⌘K → type the pane name. Close with ⌘W. Focus with ⌘+digit (the pane's position on the canvas).
