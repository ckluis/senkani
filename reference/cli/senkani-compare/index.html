senkani compare

Live

Diff two files or git revisions through the filter pipeline.

Syntax

senkani compare

Behavior

Useful for comparing agent-produced diffs to raw diffs.

Example

$ senkani compare HEAD~1 HEAD

Flags

Flag
Description
--no-filter
Disable filter for the comparison.
