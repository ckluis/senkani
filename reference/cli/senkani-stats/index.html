senkani stats

Live

Lifetime metrics from the session database. FTS5 search over tool output history.

Syntax

senkani stats [--search term]

Behavior

Default view shows lifetime tokens saved, cost saved, compliance rate, top filter rules by hit. `--search` hits the FTS5-indexed `commands.output_preview`. `--security` surfaces event counters.

Example

$ senkani stats
lifetime: 2.3M tokens saved · $14.20 · 94% compliance

Flags

Flag
Description
--search <term>
FTS5 search over command output history.
--security
Show security event counters.
--verbose
Per-project rows.
--json
JSON output.
