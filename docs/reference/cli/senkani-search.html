senkani search

Live

Symbol lookup from the tree-sitter index.

Syntax

senkani search

Behavior

Same index as `senkani_search`. Output is human-readable by default; `--json` for scripting.

Example

$ senkani search OrderRepository
src/orders/repo.ts:12  class OrderRepository

Flags

Flag
Description
--kind <k>
Filter by kind: class, function, method, etc.
--file <glob>
Restrict to matching paths.
--container <c>
Restrict to symbols inside a container.
--limit N
Max results. Default 20.

See also

  • senkani_search tool
